# Phase: slice

Goal: define the smallest verifiable delivery increment.

1. Start from one PRD acceptance outcome.
2. Include the testable behavior, required files, and explicit out-of-scope work.
3. Add todos that can be checked off by evidence.
4. Use `wiki slice create` and `wiki slice append`; do not hand-edit slice markdown.
5. A slice is ready when red/green evidence can prove it.
